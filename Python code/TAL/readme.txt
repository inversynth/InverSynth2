The following readme objective is to explain all files in this folder.

All parameters' classes are defined in the beginning of 'tal_dataset_creator.py'.
Inside Config class, you should change 'synth_plugin' path to your path file.
Using 'my_create_csv' function you create a CSV file with all configurations and then with 'read_csv_and_generate_audio' function you read the CSV and generate the sounds.
